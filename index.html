Developer Setup Guide

A streamlined automated setup configuration to empower your development environment with Claude Code and custom agent skills.

What gets installed?

The bootstrap script configures three main pillars of your interactive workspace:

🤖

Claude Code

Claude AI assistant directly embedded in your command line for coding tasks.

⚡

agent-skill

Custom SOP skills deployed at ~/.claude/skills/sop for automation.

🐙

GitHub CLI

The official gh tool for seamless command line GitHub integration.

One-time Setup

Get up and running in minutes by running the installation script and authenticating your tools.

1

Run the installation script

Run the setup automation directly in your terminal using the source command:

source <(curl -fsSL https://raw.githubusercontent.com/Siam-GS-Battery/coder-sopify/main/setup.sh)
2

Log in to GitHub

Authenticate the GitHub CLI to allow repository interactions:

gh auth login
3

Launch Claude Code

Start your interactive Claude programming session:

claude

Usage Guidelines

Maximize your productivity using custom commands and monitoring capabilities.

Running SOP Skills

Once you are inside your interactive Claude Code terminal, initiate the custom SOP routines using the slash command:

/sop

Using the Monitor Tool

For long-running tasks or log streaming, prompt Claude to use its native Monitor tool to listen for completion events and react efficiently instead of polling.

Tip: You can explicitly say: "Please use the Monitor tool to watch the deployment log."

Troubleshooting & Maintenance

Support & Troubleshooting

If you face configuration issues or installation failures, contact your system administrator for assistance.

Reverting / Uninstalling

To completely clean up and remove Claude Code and the custom agent skills, run the following command:

rm -f ~/.local/bin/claude && rm -rf ~/agent-skill ~/.claude/skills/sop

Note: To remove GitHub CLI, run: sudo apt purge gh -y
